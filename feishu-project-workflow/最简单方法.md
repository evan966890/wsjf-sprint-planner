# 🎯 最简单的方法 - 浏览器控制台运行

不需要获取Token，不需要创建配置文件，直接在浏览器中运行！

## 只需3步

### 第1步：登录并打开字段管理页面

打开这个链接并登录：
```
https://project.f.mioffice.cn/iretail/setting/workObject/story?menuTab=fieldManagement
```

### 第2步：打开浏览器控制台

按 **F12** 键，然后点击 **Console（控制台）** 标签

### 第3步：复制粘贴代码

1. 打开文件：`browser-console-script.js`
2. 复制**全部内容**
3. 粘贴到控制台
4. 按回车键

然后脚本会自动运行，3秒后开始创建字段。

## ✅ 预期结果

- 控制台会显示创建进度
- 成功后会弹出提示框
- 页面会自动刷新显示新字段

## ❓ 如果失败了

如果提示"无法获取CSRF Token"：

1. 在控制台会提示您手动输入
2. 按照"详细操作指南.md"获取Token
3. 在弹出的输入框中粘贴Token
