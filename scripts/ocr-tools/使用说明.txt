===============================================
  批量PDF转Markdown工具 - 使用说明
===============================================

问题：原batch-convert.bat出现乱码
解决：使用新的英文版本

-----------------------------------------------
  3个可用的批处理文件
-----------------------------------------------

1. quick-convert.bat (推荐 - 最简单)
   ✓ 双击运行
   ✓ 自动使用推荐设置
   ✓ 转换当前文件夹所有文件

2. run-batch-convert.bat (可选择设置)
   ✓ 双击运行
   ✓ 可选择质量模式
   ✓ 可拖拽文件夹

3. 直接使用Python脚本（最灵活）
   python batch-convert.py <文件夹路径> [选项]

-----------------------------------------------
  快速开始
-----------------------------------------------

方法1: 最简单的方式

  1. 把要转换的PDF放在一个文件夹里
  2. 复制 quick-convert.bat 到该文件夹
  3. 双击运行
  4. 完成！查看 markdown_output 文件夹

方法2: 使用可选设置

  1. 双击 run-batch-convert.bat
  2. 输入文件夹路径（或拖拽文件夹）
  3. 选择质量模式（1/2/3）
  4. 等待完成

方法3: 命令行（高级）

  打开命令行，运行：

  python batch-convert.py ./我的PDF

  或使用自定义设置：

  python batch-convert.py ./扫描文档 --resolution large --dpi 300

-----------------------------------------------
  常用命令
-----------------------------------------------

# 转换当前目录
python batch-convert.py .

# 转换指定目录
python batch-convert.py D:\我的PDF文件

# 高质量转换
python batch-convert.py ./文档 --resolution large --dpi 300

# 快速转换
python batch-convert.py ./文档 --resolution small

# 查看帮助
python batch-convert.py --help

-----------------------------------------------
  质量选项说明
-----------------------------------------------

模式1 (Quick) - 推荐
  分辨率: base (1024x1024)
  DPI: 200
  速度: 中等
  适合: 大多数文档

模式2 (High Quality)
  分辨率: large (1280x1280)
  DPI: 300
  速度: 慢
  适合: 重要文档、复杂布局

模式3 (Fast)
  分辨率: small (640x640)
  DPI: 150
  速度: 快
  适合: 简单文档、快速预览

-----------------------------------------------
  输出位置
-----------------------------------------------

默认输出目录: 输入文件夹\markdown_output\

例如：
  输入: D:\PDF文件\文档.pdf
  输出: D:\PDF文件\markdown_output\文档.md

目录结构会保持不变

-----------------------------------------------
  故障排除
-----------------------------------------------

问题1: 提示"找不到Python"
解决: 安装Python 3.10+
      下载: https://python.org

问题2: 提示"无法导入DocumentProcessor"
解决: 安装DeepSeek-OCR技能
      cd %USERPROFILE%\.claude\skills\deepseek-ocr-to-md
      python scripts\install.py

问题3: CUDA内存不足
解决: 使用更小的分辨率
      python batch-convert.py ./文档 --resolution small

问题4: 速度太慢
解决: 降低质量设置或使用更好的GPU
