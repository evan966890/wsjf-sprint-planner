========================================
  📋 如何检查DeepSeek-OCR安装状态
========================================

方法1: 简单检查（PowerShell中运行）
--------------------------------------

wsl -d Ubuntu bash -c "ls ~/deepseek-env/bin/python3 2>/dev/null && echo '✓ Python环境已创建' || echo '⏳ 还在安装'"

结果:
  "✓ Python环境已创建" → 环境创建完成
  "⏳ 还在安装" → 继续等待


方法2: 完整验证（PowerShell中运行）
--------------------------------------

wsl -d Ubuntu bash -c "source ~/deepseek-env/bin/activate 2>/dev/null && python3 -c 'import vllm; print(\"✓ DeepSeek-OCR安装成功\")' || echo '⏳ 还在安装vLLM'"

结果:
  "✓ DeepSeek-OCR安装成功" → 全部完成！
  "⏳ 还在安装vLLM" → 继续等待


方法3: 查看WSL2进程（PowerShell中运行）
--------------------------------------

wsl -d Ubuntu bash -c "ps aux | grep -E 'apt-get|pip|python' | wc -l"

结果:
  数字 > 5 → 有安装进程在运行
  数字 <= 5 → 可能已完成或未开始
